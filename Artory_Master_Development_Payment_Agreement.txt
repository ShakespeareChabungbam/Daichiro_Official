SECTION 1 — SCOPE OF WORK
================================================================================

1.1  Technology Stack:
     Frontend  : HTML, CSS, JavaScript
     Backend   : Python, Flask
     Database  : PostgreSQL
     AI Engine : Google Gemini AI (Vision + Text Generation)
     Server    : AWS EC2 (Ubuntu), Nginx, Gunicorn

1.2  Deliverables:
     (a) Full-stack web application deployed at daichiro.in
     (b) Public website — Booking, About, SkillNest, Sunday Programs
     (c) Admin Dashboard
     (d) Employee Staff Portal (role-based, permission-controlled)
     (e) Client Assessment Portal (self-destructing credentials)
     (f) AWS EC2 deployment with Gunicorn + systemd
     (g) Nginx reverse proxy with HTTPS / SSL
     (h) PostgreSQL database schema and backup scripts
     (i) Sitemap, robots.txt, SEO structure
     (j) Maintenance page for downtime

1.3  Features:
     (a) Appointment Booking — Phone validation, package selection,
         price-locking, booking open/close control
     (b) AI Career Assessment (5-Step) — Auto-save, Gemini AI analysis,
         PDF reports, OCR photo scan via Gemini Vision
     (c) SkillNest Admissions — Seat cap, open/close toggle, admin workflow
     (d) Invoice & Email — HTML invoice emails via Gmail SMTP,
         password reset emails with signed tokens
     (e) Employee Permissions — Granular role-based access control
     (f) Admin Panel — Full control over all data and settings
     (g) Site Config Store — All settings manageable from admin panel
     (h) Background Automation — Client auto-expiry (3 hrs),
         assessment archival (90 days), appointment archival (90 days)
     (i) Security — CSRF, rate limiting, HSTS, CSP, XSS headers,
         bcrypt hashing, secure sessions
     (j) Developer Credit Portal — ArtoryX's branded credit management
